01. Клонировать репозиторий проекта и перейти в папку проекта
02. Убедиться, что установлен Python версии 3.9 или выше и он доступен в PATH
03. Проверить структуру проекта: в корне должны быть app.py, requirements.txt, папка bot/, файл run_all.bat
04. Создать файл .env в корне проекта, если он отсутствует
05. Заполнить файл .env переменными BOT_TOKEN, API_URL, CHAT_ID
06. Добавить Telegram-бота в нужную группу или супергруппу
07. Выдать боту права на отправку сообщений в группе
08. В BotFather отключить Privacy Mode для бота
09. Убедиться, что файл run_all.bat находится в корне проекта
10. Запустить проект двойным кликом по файлу run_all.bat
11. Дождаться, пока батник создаст виртуальную среду и установит зависимости
12. Убедиться, что FastAPI-сервер запущен
13. Убедиться, что Telegram-бот запущен без ошибок
14. Проверить работу, отправив сообщение в Telegram-группе
15. Убедиться, что сообщение сохранилось в базе данных и появилось в группе
16. Для остановки проекта закрыть оба консольных окна
